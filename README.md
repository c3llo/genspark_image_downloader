# genspark_image_downloader

1. 크롬 확장 프로그램 
   chrome://extensions/

2. 개발자 모드 켜기

3. 압축해제된 확장 프로그램 로드

4. 해당 소스를 특정 폴더에 다운로드 받은후, 해당 폴더 선택

5. AI Asset Master Collector가 생겼는지 확인

6. 젠스파크 에셋에서 새로고침 한번 하고, 우측상단에 확장프로그램 표시

7. 이미지 체크박스에 마우스를 가져다 두고, ALT + 클릭하면 숫자가 증가되면서 다운로드 목록에 추가됨

8. 다운로드 규칙을 설정후, 다운로드 버튼을 누르면 규칙에 따라 이미지가 다운로드 됨
